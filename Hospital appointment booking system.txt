#  Hospital Appointment Booking System

## Project Overview

The Hospital Appointment Booking System is a menu-driven application developed using **Python** and **MySQL**. The project helps in managing hospital appointments by providing separate functionalities for administrators and patients.
The system allows administrators to manage doctors, patients, and appointments, while patients can register, log in, and book appointments with available doctors.

---

##  Objectives

- Automate the hospital appointment booking process.
- Store hospital records permanently using MySQL.
- Provide separate modules for Admin and Patient.
- Perform CRUD operations using Python and SQL.
- Build a simple, user-friendly menu-driven application.
